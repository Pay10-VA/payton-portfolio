My Projects

Mobile App
Twitter Clone

Skills used: Java, Amazon Web Services

Created a native Android replica of Twitter. The front-end was created in java using the Model-View-Presenter architecture and implements the observer and template method design patterns.

The back-end relied heavily on Amazon Web Services. Lambda and Gateway were used to handle communications to and from the client, and DynamoDB and S3 were used for storing user data and profile pictures, respectively. In order to handle updating a user's follower's feed(given a large amount of followers), I used SQS to asyncronously update all the user's follower's feeds.
