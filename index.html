My Projects

Mobile App
Twitter Clone

Skills used: Java, Amazon Web Services

Created a native Android replica of Twitter. The front-end was created in java using the Model-View-Presenter architecture and implements the observer and template method design patterns.

The back-end relied heavily on these Amazon Web Services: Lambda, API Gateway, DynamoDB, S3, and SQS.

Mobile App
Family Map

Skills used: Java, Amazon Web Services

Created the back-end and front-end of a multi-view Android application that allows user's to view their family history data. The application features an interactive map with markers that displays important events in the user's family history. User's can toggle with the settings page to filter out events from showing on the map.

Mobile App
Work Together

Skills used: JavaScript, React Native, Expo

Used the React-Native framework to create the front-end of a mobile app that allows small businesses to build their networks and send and receive referrals.

Web App
Mock Vaccine Finder

Skills used: HTML, CSS, JavaScript, Vue CLI, MongoDB, Express

I used Vue CLI to develop a multi-view web application (both front and back end) that helps users locate Covid-19 Vaccination sites in the state of Nevada. The application utilizes user authentication and MongoDB to store all user data, and a rest API that supports CRUD operations on that database.

Web App

Skills used: HTML, CSS, JavaScript, External API
